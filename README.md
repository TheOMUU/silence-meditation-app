 🧘 Silence - Meditation App

An AI-powered meditation app that detects user emotions and suggests calming music.

🚀 Features
- 🎧 Meditation music (528Hz, deep sleep, healing sounds)
- 🤖 Emotion detection using AI
- 📊 User history tracking
- 🌐 Web-based UI inside Android app

🛠️ Tech Stack
- Java (Android)
- HTML, CSS, JavaScript
- SQLite Database
- Emotion Classification Model

📂 Project Structure
- app/src/main/java → Core logic
- assets → UI + music
- res → Android UI
